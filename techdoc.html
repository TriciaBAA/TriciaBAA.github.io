Introduction to CSS Flexbox

CSS Flexible Box Layout, commonly known as Flexbox, is a layout model designed for the arrangement of elements in a responsive and efficient way in one dimension, either horizontally or vertically.

Flexbox provides an efficient way to layout, align, and distribute space among items in a container even when their size is unknown or dynamic. It is particularly useful in creating complex layouts and aligning elements within a container.

Flex Container

In Flexbox, the container is referred to as the flex container, and its direct children as flex items. To enable Flexbox layout for an element, you need to set its display property to flex or inline-flex.

For example, to create a flex container, you can use the following CSS:

.flex-container { display: flex; }
Flex Items

Flex items are the immediate children of a flex container. They are laid out along the main axis of the flex container. By default, flex items will try to fit into a single line, but they can also wrap onto multiple lines if needed.

To define an element as a flex item, you don't need to use any additional CSS. Any direct child of a flex container becomes a flex item automatically.

Flex Direction

Flex direction determines the primary axis of the flex container, along which flex items are laid out. It can be set to row, row-reverse, column, or column-reverse.

For example, to create a flex container with a column layout, you can use the following CSS:

.flex-container { display: flex; flex-direction: column; }
Justify Content

Justify content defines how flex items are aligned along the main axis of the flex container. It can be used to distribute space between and around flex items.

Common values for justify content include flex-start, flex-end, center, space-between, and space-around.

For example, to justify content to the center within a flex container, you can use the following CSS:

.flex-container { display: flex; justify-content: center; }
Align Items

Align items defines how flex items are aligned along the cross axis of the flex container. It can be used to align items at the start, end, center, or stretch them to fill the container.

Common values for align items include flex-start, flex-end, center, baseline, and stretch.

For example, to align items to the center vertically within a flex container, you can use the following CSS:

.flex-container { display: flex; align-items: center; }
Align Content

Align content defines how multiple lines of flex items are aligned along the cross axis of the flex container. It only has effect when flex items wrap onto multiple lines.

Common values for align content include flex-start, flex-end, center, space-between, space-around, and stretch.

For example, to align content to the center vertically within a flex container with wrapping enabled, you can use the following CSS:

.flex-container { display: flex; flex-wrap: wrap; align-content: center; }
Flex Grow

Flex grow determines how much a flex item grows relative to other flex items in the flex container when there's extra space available. It's specified using a unitless value, which represents the proportion of the available space the flex item should take up.

By default, flex grow is set to 0, meaning flex items won't grow. Setting it to 1 makes the item grow to fill the available space.

For example, to make a flex item grow relative to others, you can use the following CSS:

.flex-item { flex-grow: 1; }
Flex Shrink

Flex shrink determines how much a flex item shrinks relative to other flex items when there's not enough space available. It's specified using a unitless value, which represents the proportion of the available space to distribute among the flex items.

By default, flex shrink is set to 1, allowing flex items to shrink proportionally. Setting it to 0 prevents the item from shrinking.

For example, to prevent a flex item from shrinking, you can use the following CSS:

.flex-item { flex-shrink: 0; }
Flex Basis

Flex basis defines the initial size of a flex item before distributing the remaining space. It can be set as a length value such as pixels or percentages, or as the keyword auto to use the item's natural size.

By default, flex basis is set to auto, meaning the browser calculates the size based on the content.

For example, to set the initial size of a flex item, you can use the following CSS:

.flex-item { flex-basis: 100px; }
Flex

Flex is a shorthand property for setting the flex grow, flex shrink, and flex basis properties in one declaration. It allows you to specify how a flex item will grow, shrink, and behave in relation to the other flex items in the flex container.

The values for the flex property are as follows: flex-grow, flex-shrink, and flex-basis.

For example, to set the flex properties for a flex item, you can use the following CSS:

.flex-item { flex: 1 0 auto; }
Order

Order determines the order in which flex items appear within the flex container, regardless of their source order in the HTML. It's specified using an integer value, where lower values come first.

By default, flex items have an order of 0, meaning they appear in the source order. Negative values are allowed.

For example, to change the order of a flex item, you can use the following CSS:

.flex-item { order: 2; }
Align Self

Align self allows you to override the align items property for individual flex items within the flex container. It's used to control the alignment of a single flex item along the cross axis.

Common values for align self are the same as align items: flex-start, flex-end, center, baseline, and stretch.

For example, to align a specific flex item to the end within a flex container, you can use the following CSS:

.flex-item { align-self: flex-end; }
